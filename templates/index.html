FAQs
What is Piyu-V?
This is a research-oriented and data-driven project aimed at building a hybrid model of machine learning algorithms such as ST-GCN and LSTM to optimize public transportation. It predicts demand and recommends strategies for route optimizations by analyzing historical data. Moreover, it integrates and tailors several emerging technologies in spatial-temporal modeling to align the goals of urban planning with sustainability, mobility, and efficiency for long-term solutions at a system level—which would cater to different stakeholders, such as policymakers, transit operators, and commuters.
What file types does Piyu-V accept?
Piyu-V only accepts files that are in .csv formats for trip data.
What should be the contents of the trip dataset?
In order to achieve Piyu-V's full capacity, the trip dataset to be uploaded should contain the following attributes: date, PULocationID, day_of_week, and passenger_count. For more information, please refer to Piyu-V-trip-data-guide.
Does Piyu-V support multiple file uploads?
Yes, just ensure that the files have the corresponding attribute names.
Does Piyu-V perform data pre-processing?
Currently, Piyu-V does not have a data pre-processing feature. This means that whatever you upload will be directly used by the system for model execution without any automatic data cleaning, transformation, or formatting. The system relies on the uploaded data being in the correct format for the model to run effectively.
